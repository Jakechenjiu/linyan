# 灵砚 LingYan — 产品简报

## 产品定位
灵砚是一个 AI 驱动的创作工作台，面向网文作者和内容创作者，提供从构思、写作到发布的一站式工具链。

## 目标用户
- 网文作者（玄幻、仙侠、都市等类型小说）
- 自媒体内容创作者（公众号、短视频脚本）
- 需要知识管理和思维整理的创作者

## 核心模块

### 1. 星辰 Star — 小说创作系统
- 小说管理（创建、分类、状态追踪）
- 世界观设定（力量体系、地理、势力、规则）
- 角色系统（主角/反派/配角，含金手指、成长弧线）
- 大纲系统（卷/章节结构，CBN/CPNs/CEN 节点）
- 章节编辑器（正文写作、字数统计、事实快照）
- 写作日志（每日字数追踪）

### 2. 光子 Photon — 内容发布系统
- 内容编辑器（富文本，多平台适配）
- 工作室（AI 生成媒体、脚本优化）
- 批量操作（批量生成/发布）
- 内容日历（排期管理）
- 数据分析（阅读量、互动数据）
- 模板系统（文章/视频脚本模板）
- 剪映导出

### 3. 笔记 Notes — 知识管理系统
- 双向链接笔记
- 知识图谱可视化
- 标签系统

### 4. 脑图 Mindmaps — 思维整理
- Markdown 驱动的脑图
- 节点计数

### 5. 万象 Wanxiang — 模拟推演
- 多智能体模拟
- 场景分析
- 历史记录

### 6. 视频 Video — 短视频制作
- 分镜脚本
- AI 生成视觉素材
- 配音合成
- 多平台适配（抖音等）

### 7. 实验室 Lab — 实验性功能
- 故事创作工具
- 脑图实验

## 技术栈
- **前端**: Next.js 16 + React 19 + Tailwind CSS v4
- **后端**: Next.js API Routes + Hono
- **数据库**: PostgreSQL + Prisma 7 ORM
- **认证**: NextAuth v5
- **3D**: Three.js + React Three Fiber
- **状态管理**: Zustand
- **AI**: 用户自带 API Key（DeepSeek/OpenAI/Anthropic + DashScope 通义万相）

## 商业模式
- 用户自带 API Key，平台提供工具
- 核心价值在工作流整合，而非 AI 能力本身
