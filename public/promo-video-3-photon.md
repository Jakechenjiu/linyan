## 录屏准备

- [ ] **提前创建一个完整项目**，话题"三亚旅行攻略"，脚本和分镜已生成
- [ ] **提前生成好至少 1 个分镜的视频**（DashScope API Key 配好，跑一次完整流程）
- [ ] 确保"导出剪映"功能可用
- [ ] 准备一个"正在生成"的状态画面（如果有正在跑的更好，否则录之前生成一个预览用的）

## 录屏操作

1. 从光子发布 project studio 页面开始（不是首页，是已经创建好的项目）
2. 录制输入话题 → AI 生成脚本的流程（如果已有项目，可以新建一个录）
3. 展示分镜卡，点进一张，highlight 脚本和画面描述
4. 回到时间线视图，展示"生成素材"按钮
5. 展示生成的视频：点播放，让视频充满屏幕
6. 展示"导出剪映"
7. 停止录屏

> 关键：这个视频的核心是 22-28s 那段成品展示。提前生成好的视频质量决定这个视频的冲击力。
